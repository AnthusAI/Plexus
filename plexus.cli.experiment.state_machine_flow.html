plexus.cli.experiment.state_machine_flow module

State Machine-Based Conversation Flow Manager

This module provides an intelligent conversation state machine that: 1. Tracks tool usage patterns and progress 2. Uses predicate logic for state transitions 3. Provides dynamic prompt templates based on current state 4. Enables proper investigation before forcing hypothesis generation

Manages conversation flow using a state machine approach based on tool usage.

States advance based on actual tool usage patterns rather than just message counts, ensuring the AI has gathered sufficient information before being pushed to create hypotheses.
