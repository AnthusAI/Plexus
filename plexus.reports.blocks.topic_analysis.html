class plexus.reports.blocks.topic_analysis.TopicAnalysis(config: Dict[str, Any], params: Dict[str, Any] | None, api_client: PlexusDashboardClient)

Bases: BaseReportBlock

DEFAULT_NAME: str | None = 'Topic Analysis'

Performs topic analysis on transcript data and attaches resulting artifacts.

This block orchestrates transcript transformation and BERTopic analysis, similar to the plexus analyze topics CLI command.
