plexus.analysis.opensearch_client module

OpenSearch client for Vector Topic Memory.

Thin adapter wrapping opensearch-py with AWS SigV4 auth. Supports health check, bulk indexing with blue/green alias swap, and KNN search for nearest cluster assignment.
