plexus.scores.Score module

class plexus.scores.Score.Score(**parameters)

Bases: ABC, PythonModel

Abstract base class for a score.

This class provides a framework for implementing different scoring models. It includes methods for parameter validation, report directory management, and configuration recording. Subclasses must implement the load_context and predict methods.
