# Diagram Standards

Use `documentation/diagrams/palette.excalidraw` as the single shared palette for Excalidraw diagrams in this repository.

## Workflow

1. Open `documentation/diagrams/palette.excalidraw`.
2. Duplicate the elements you need into your target diagram.
3. Save the finished diagram in the topic folder that owns it.
4. Keep diagram-specific content in the diagram file instead of adding it back to the shared palette.

## Naming And Location

- Put shared reusable Excalidraw assets at the top level of `documentation/diagrams/`.
- Put finished diagrams in the topic folder that matches their subject area.
- Use descriptive diagram file names that reflect the concept being explained.
- Keep optimizer procedure diagrams in `documentation/diagrams/optimizer-multi-agent-procedure/`.

## Palette Contents

The shared palette should contain only reusable visual building blocks such as:

- layout containers
- process boxes
- arrows and connectors
- label treatments
- color semantics that are meant to stay consistent across diagrams

Do not turn the shared palette into a library of topic-specific diagrams or optimizer-specific content.
